# Figures

This folder contains figures generated from the final experiments.

## Main figures

| File                              | Description                                              |
|-----------------------------------|----------------------------------------------------------|
| `fig10_dataset_distribution.png`  | Class distribution of the dataset (134 cl. vs 99 q.).    |
| `methodology_figure.png`          | End-to-end fine-tuning pipeline.                         |
| `fig8_mean_metrics.png`           | Cross-backbone metric comparison (accuracy / F1 / AUC).  |
| `fig1_confusion_matrix.png`       | Pooled confusion matrix.                                 |
| `fig5_roc_curve.png`              | ROC curve comparison.                                    |

These figures are used in the final report. They are regenerated by
running:

```bash
python scripts/generate_figures.py
```

## Additional figures

The folder also contains per-backbone SVG variants
(`fig1_confusion_matrix_<short>.svg`,
`fig2_fold_distribution_<short>.svg`, `fig3_roc_curve_<short>.svg`) and
three-way comparison figures (`fig_cross_model_comparison.svg`,
`fig_roc_overlay.svg`, `fig_paired_per_fold.svg`) generated during the
full training run by `scripts/run_training.py`.
