\section*{Abstract}
This report describes a controlled three-backbone comparison
(\texttt{roberta-base}, \texttt{microsoft/codebert-base},
\texttt{microsoft/graphcodebert-base}) for binary classification of
software-engineering bug reports as classical or quantum. With identical
architecture, parameter count, and 5-fold $\times$ 5-seed cross-validation
protocol, the comparison isolates the effect of pretraining data on
this downstream task.
